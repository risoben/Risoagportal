ICON STYLING CONSISTENCY:

All Icons:
- Color: Original asset color OR override to #0F429F (Riso Blue)
- Background Container: #F0F4FF (only for large 48x48px versions)
- Border-Radius: 8px (background container only)
- Padding: 8px (background container only)
- Opacity: 1.0 (default), 0.6 (disabled/inactive state)
- Hover: Slight scale (1.05x) on interactive elements

Small Icons (16x24px):
- No background container
- Direct color: #0F429F or asset original
- Hover: Opacity 0.8

Medium Icons (32x32px):
- Optional background container
- Padding: 6px
- Border-Radius: 6px

Large Icons (48x48px):
- Required background container (#F0F4FF)
- Padding: 8px
- Border-Radius: 8px
- Shadow: subtle (0 2px 4px rgba(0,0,0,0.1))
